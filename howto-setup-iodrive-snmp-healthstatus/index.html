How to setup ioDrive's SNMP

If we want to know the health status of ioDrive/ioDrive2, how should I do it? Generally, we use fio-status command. However, you must log in to a server to user fio-status each time. fio-status is incovenient for hurrying time and a periodical monitoring.

In such a case, I recommend to monitor health status of ioDrive using SNMP.

rpm -ivh ./fio-snmp-agentx-1.x.x86_64.rpm
